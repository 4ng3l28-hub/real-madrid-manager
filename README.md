# real-madrid-manager
Proyecto-Sistema de gestión del Real Madrid desarrollado en PHP, MySQL, JavaScript y APIs REST.
Sistema web desarrollado en PHP y MySQL para la gestión deportiva del Real Madrid.

## Funcionalidades

- Inicio de sesión
- Dashboard
- Gestión de jugadores
- Gestión de fichajes
- Gestión de ventas
- Gestión de partidos
- Consumo de APIs REST
- Administración de imágenes
- Base de datos MySQL

## Tecnologías

- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
- REST API
- XAMPP
- GitHub
- AWS (próximamente)

## Instalación

1. Clonar el repositorio.
2. Importar `database/real_madrid_manager.sql`.
3. Configurar `config/conexion.php`.
4. Ejecutar desde XAMPP.
